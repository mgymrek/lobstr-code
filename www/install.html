Installation from precompiled binaries

  1. Download the lobSTR package.
  2. Unzip the package to $PATH_TO_LOBSTR.
  3. Change to the $PATH_TO_LOBSTR directory.
  4. Copy the three executable binary files to a place in your $PATH. e.g.
      5. cp bin/lobSTR /usr/bin
      cp bin/allelotype /usr/bin
      cp bin/lobSTRIndex /usr/bin
For usage instructions see usage.

Installation from source

lobSTR source code can be obtained from our github repository: https://github.com/whitehead/lobstr-code.git
  1. Make sure all required packages are installed and in the user's $PATH (see below for a list of requirements).
  2. Clone the lobSTR git repository: git clone https://github.com/whitehead/lobstr-code.git $PATH_TO_LOBSTR
  3. Change to the $PATH_TO_LOBSTR directory.
  4. Type the following commands:
    • ./reconf
    • ./configure
    • make
    • make install
If this returned no error messages then lobSTR was properly installed. For usage instructions see usage.
